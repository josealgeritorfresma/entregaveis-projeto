# Checklist de Go-live

## Técnico
- [ ] CI verde (lint/test/build)
- [ ] SAST/dependências sem vulnerabilidades críticas
- [ ] Observabilidade ativa (logs/métricas/alertas)
- [ ] Backup/restore validado (quando aplicável)
- [ ] Rollback testado (quando possível)

## Negócio
- [ ] Homologação aprovada
- [ ] Comunicação aos usuários realizada
- [ ] Suporte preparado (SLA/contatos)
