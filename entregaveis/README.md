# Pacote de Entregáveis (SDLC) — Node.js + React + TypeScript

Este diretório contém templates (em PT-BR) para as etapas:
1. Levantamento e Análise de Requisitos
2. Projeto e Arquitetura
3. Desenvolvimento (Implementação)
4. Testes e Validação
5. Implantação (Deploy)
6. Acompanhamento e Manutenção

## Como usar
- Preencha os campos entre <> e as seções marcadas como "A definir".
- Ajuste os artefatos conforme o tipo de produto (web app, API, BFF etc.).

## Convenções sugeridas
- Requisitos: RF-###, RNF-###
- Casos de teste: CT-###
- ADRs: ADR-###
- Itens de mudança: CHG-###
